Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects, allowing developers to manipulate the content, structure, and styles of web pages programmatically.

How do you select on element from the DOM?

To select an element from the DOM, i can use: 1. getElementById('id'): Selects by ID. 2. getElementsByClassName('class'): Selects by class name. 3. getElementsByTagName('tag'): Selects by tag name. 4. querySelector('selector'): Selects the first matching element using a CSS selector. 5. querySelectorAll('selector'): Selects all matching elements using a CSS selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM that involves using a single event listener on a parent element to manage events for multiple child elements. Instead of attaching individual listeners to each child, you attach one listener to the parent, which then captures events that bubble up from the children.

How do you manipulate an elements attributes and styles using the DOM?

I can manipulate an element's attributes and styles in the DOM using the following methods: ### Attributes 1. **Get an Attribute**: Use `getAttribute('attributeName')`. 2. **Set an Attribute**: Use `setAttribute('attributeName', 'value')`. 3. **Remove an Attribute**: Use `removeAttribute('attributeName')`. 4. **Direct Property Access**: Access some attributes directly as properties (e.g., `element.id`, `element.className`). ### Styles 1. **Inline Styles**: Use `element.style.property = 'value'` to set styles directly. 2. **Multiple Styles**: Use `Object.assign(element.style, { /* styles */ })` to apply multiple styles at once. 3. **Remove a Style**: Use `element.style.removeProperty('propertyName')` to remove specific styles.
